# Troubleshooting

## Colors aren't displaying correctly

If you see a warning at the top of the sidebar about "Object Color Type is not enabled", click **Open Recommended Settings**. Home Builder requires the viewport shading color type to be set to **Object** for colors to display correctly.

## Add-on doesn't appear after installation

- Make sure you're using **Blender 5.0** or later
- Check **Window → Toggle System Console** for error messages
- Try reinstalling the add-on

## Cabinets aren't snapping to walls

- Make sure the wall was created with Home Builder's **Draw Walls** command (not a manually created mesh)
- Check that you're clicking along a wall surface during placement
